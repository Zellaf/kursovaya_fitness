Добро пожаловать в информационно-справочную систему фитнес-клуба

Золотов Дмитрий Витальевич ПИ21-4

Целью описываемой работы является разработка клиент-серверного приложения «Фитнес» на языке программирования Java, содержащая в себе серверную часть с использованием фреймворка Spring Boot. Решение использует модель MVC, согласно которой управляющая логика программы разделяется на отдельные компоненты. За счёт применения Java и JVM решение является кроссплатформенным.
